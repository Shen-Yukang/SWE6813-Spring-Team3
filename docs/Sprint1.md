# Sprint1.md

## Sprint Goal
Deliver a working prototype of the matchmaking platform.

Users must be able to:
- register
- create a profile
- request matchmaking results

## Stories

1. Project Setup
Initialize repository, backend server, and basic configuration.

2. User Authentication
Implement registration and login.

3. Player Profile
Allow users to store skill and behavior attributes.

4. Basic Matchmaking
Return top recommended players based on skill and behavior.

5. Minimal UI
Simple interface to trigger matchmaking.

6. Testing Baseline
Add initial unit tests and one BDD scenario.

## Deliverables

Working prototype
10+ unit tests
1 BDD scenario
Scrum artifacts (board, burndown, daily logs)
